THE NBA TALK
The nba talk is a social media application targeted for nba fans. During this project I have implemented advances features that a typical social media app have, such as user being able to post something, react to another user post with liking, disliking, commenting. The app also have a blog where designated writer have written articles about the NBA.
Technologies used: Python/Django, Angular, HTML, CSS, PosgresSQL
